# Reinforcement Learning for Algorithmic Trading

This project develops a Reinforcement Learning agent capable of learning trading strategies for financial assets using PPO (Proximal Policy Optimization).

## Tech Stack

- Python
- Gymnasium
- Stable-Baselines3
- PPO
- Pandas
- NumPy
- Matplotlib
- YFinance
